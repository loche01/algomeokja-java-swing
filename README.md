# 알고먹자 - 식단·운동 관리 프로그램 (Java Swing, MySQL)

알고먹자는 사용자가 식단과 운동을 기록하고, 하루 동안 섭취한 칼로리와 영양소, 운동 기록, 체중 목표를 한곳에서 관리할 수 있는 Java Swing 기반 데스크톱 프로그램입니다.

아침·점심·저녁 등 식사별로 음식을 검색해 식단을 기록할 수 있으며, 운동 종류와 운동 시간을 입력해 운동 기록을 저장할 수 있습니다. 저장한 기록은 날짜별로 확인할 수 있고, 목표 체중을 설정하여 현재 진행 상황을 확인할 수 있습니다.

2025년 4인 팀 프로젝트로 개발했으며, 프로젝트 종료 후 기존 프로젝트를 다시 실행하고 분석하면서 실행 환경, 보안, 데이터 처리, 기능 오류 및 UI/UX를 개인적으로 개선했습니다.

## 🛠️ 사용 기술

### Application

- **Java 21** - 애플리케이션 기능 구현
- **Java Swing** - 데스크톱 GUI 구성

### Database

- **MySQL** - 회원, 음식, 식단, 운동, 목표, 공지사항 데이터 관리

- **DAO Pattern** - 화면 로직과 데이터베이스 접근 코드 분리
- **PBKDF2** - 회원 비밀번호 해싱 및 검증

### Development Environment

- **Eclipse**
- **JDK 21**
- **Git**
- **GitHub**

## 🚀 실행 환경 및 실행 방법

### 필요 환경

| 항목 | 기준 |
|---|---|
| JDK | Eclipse Temurin JDK 21 |
| IDE | Eclipse IDE for Java Developers |
| DB | MySQL |
| JDBC Driver | MySQL Connector/J |
| 프로젝트 형식 | Eclipse Java Project |

### 실행 순서

1. 저장소를 내려받습니다.
2. Eclipse에서 `Existing Projects into Workspace`로 프로젝트를 가져옵니다.
3. MySQL Connector/J를 내려받아 `lib/mysql-connector-j.jar`로 배치합니다.
4. MySQL에서 `sql/schema.sql`을 실행하여 데이터베이스 스키마를 구성합니다.
5. `src/db.properties.example`을 복사하여 `src/db.properties`를 생성하고 로컬 DB 접속 정보를 입력합니다.
6. Eclipse에서 JDK 21이 적용되어 있는지 확인합니다.
7. `src/main/MainFrame.java`를 Java Application으로 실행합니다.

> 실제 DB 접속 정보가 포함된 `src/db.properties` 파일은 Git에 포함되지 않습니다.
